## Install
Checkout the repo and run `composer install`

## Run

##### Note: The task have two solutions - OOP and one-file non-OOP (kinda POC)

To run OOP solution execute:  
`php analyzer.php`

To run OOP solution execute:  
`php analyzer-poc.php`

## Test

Execute `./test.sh`

Included tests:
- test for PSR2
- test for PHP mess (phpmd)
- Unit tests
